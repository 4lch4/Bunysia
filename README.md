# @4lch4/bunysia

This repo is a demo of how to use Bun, Elysia, and Doppler, to create a simple API with TypeScript.

To install dependencies:

```bash
bun install
```

To run:

```bash
bun start
```

To run with reloading:

```bash
bun dev
```

## Inclusions

This demo also shows how to use the following dependencies:

- Bun
- Elysia
- Doppler
- Upstash
- Redis
- Prettier
- p-queue
